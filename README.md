# Screen 演出プログラム（pygame）

自動卓球装置向けの  
**フルスクリーン演出表示プログラム** です。

キーボード入力に応じて  
画面表示・背景演出・パーティクル・音楽再生  
を切り替えます。

Arduino 側（LED & 回転灯）と  
同じキー操作を使うことで、演出の同期が可能です。

---

## 開発環境

windows 11

## 使用技術

- Python 3
- pygame
- フルスクリーン表示
- ピクセルフォント演出
- パーティクル / グリッチ演出

---

## 起動方法

python main.py

---

## 必要ライブラリ

pip install pygame

---

## 操作方法

キー | モード  
---- | ----  
0 | WAITING  
1 | EASY  
2 | HARD  
3 | EXTRA  
ESC | 終了  

---

## モード説明

### WAITING
- 暗めの背景
- 待機状態の表示

### EASY
- シアン系カラー演出
- 穏やかなパーティクル表示

### HARD
- オレンジ系カラー演出
- 緊張感のあるビジュアル

### EXTRA
- 赤系カラー演出
- 画面揺れ
- グリッチ演出強化
- BGM ループ再生
